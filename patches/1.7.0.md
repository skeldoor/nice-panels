# 1.7.0

## Loot Calculator

- New probability mode that shows the real percentage chance of getting at least one drop, colour coded by likelihood
- Section dropdown to filter monsters with multiple sub-tables like "Weapons and armour" or "Seeds"
- Sort options for wiki order, rarest first, most common first, highest expected, and alphabetical
- Size presets for full screen, HD, and small layouts
- Adjustable fixed height, 1-6 column layouts, and custom title and background colours
- Panel is now centred on the page and no longer clips at wider sizes

## Icon Creator

- New checkbox to toggle the drop shadow on or off

## Info Box

- Added RS Plain, RS Bold, and RS UF font options
- New checkbox to hide the item holder image
- Custom border colour picker

## Bank View

- Fixed item count text drifting out of position at non-default scales

## Dialog Panel

- Custom chathead support via file upload or image URL
- NPC name can now be set independently of the chathead source
