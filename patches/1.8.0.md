# GE Purchase Panel

New panel for creating Grand Exchange offer slots.

## New Panel: GE Purchase Panel

- **Custom item icons** with the classic OSRS item shadow
- **Three text fields** for the offer type, item name, and price
- **Colour pickers** inline with each text field so you can tweak the look
- **Multi-line item names** using \n for longer item text
- **Pixel-perfect export** that downloads as a clean PNG
