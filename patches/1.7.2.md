# Orb Panel

A brand new panel for generating OSRS minimap orb indicators.

## New Panel: Orb Panel

- **Hitpoints, Prayer, Run Energy** orbs with on/off variants for Prayer and Run
- **Poison, Venom, and Skulled** orb types included
- **Fill level** control that crops the orb colour from the top to show how full it is
- **Value text** (0â€“100) with colour that smoothly transitions from red to yellow to green
- **Icon scale** setting so you can resize the icon on the orb
- **Custom mode** where you can paste any icon URL, pick an orb colour with a colour slider, and it tints a base orb to match
